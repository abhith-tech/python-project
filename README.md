# Billing Management System

~ Features :
   * Product listing with prices
   * Add products to cart with quantity
   * Cart view with item-wise total
   * Automatic subtotal, tax (5%), and grand total calculation
   * Bill receipt generation in a separate window
   * User-friendly Tkinter GUI
  
~ Technologies Used :
   *Python 3.x
   *Tkinter(GUI)
   *ttk(Styled widgets)

~ Project Structure :
   Billing-Management-System/
   │
   ├── billing_system.py  
   └── README.md           

~ Sample Bill Format :

   ----------------- BILL RECEIPT -----------------
   Product               Qty     Price      Amount
   Pen                     2        10          20
   Notebook                1        50          50

   Subtotal: 70.00
   Tax (5%): 3.50
   Grand Total: 73.50
